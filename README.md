# Echo Local Business Manager - Complete Documentation

## 📱 Overview
This is a **single-page Progressive Web App (PWA-like)** for managing a local business (phone repairs + customer services). Built with vanilla **HTML/CSS/JS** + **Firebase Firestore** for cloud backup. Fully **offline-first** (LocalStorage primary, Firebase sync optional). **Mobile-responsive** with modern glassmorphism design.

**Live Demo**: Open `index.html` in any browser.

## 💾 Data Flow (Offline-First)
```
Form Submit → moduleArray.push() → saveData('echo_clients', JSON) → renderTable()
↓ (if online)
Firebase batch: delete old → set new docs
```
- **Storage keys**: `echo_clients`, `echo_repairs`, etc.
- **Export**: All data → `echo-backup.json`
- **Sync**: `syncAll()` button (disabled offline)
